# Battleship (Java)

A simple 2-player console Battleship game built in Java. Players place ships on a 5x5 grid and take turns attacking until one player sinks all opponent ships.

## Features

* 2-player turn-based gameplay
* 5x5 grid using 2D arrays
* Input validation (bounds + duplicate shots)
* Hit/miss tracking with separate boards
* Game loop with win detection

## Structure

* `Battleship.java` – main driver class containing full game logic

## Concepts Used

* Arrays (2D grids)
* Loops and nested iteration
* Conditionals and input validation
* State tracking across turns
* Console I/O using Scanner

## Context

Developed as part of the edX course **CS1331 (Georgia Tech)**, demonstrating practical implementation of core programming concepts and game logic design.
